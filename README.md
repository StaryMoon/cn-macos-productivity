## Overview

整理 Mac 上最常用的启动器、窗口管理、包管理、终端、截图、播放器和虚拟化工具。

本仓库只保留与主题直接相关、入口稳定、说明清晰的资源。优先收录官方文档、主流开源仓库、长期可访问的产品页面和常用工具链。

## Recommended Path

1. 先安装 Homebrew，再配置窗口管理和终端工具。
2. 启动器建议固定一个主力工具，避免重复配置。
3. 开发者优先配置终端和容器环境。

## Notes

- 避免安装来源不明的破解工具。
- 权限类工具需关注辅助功能、屏幕录制和自动化权限。
